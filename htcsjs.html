HTML/CSS/JS

What Is HTML?

HTML or HyperText Markup Language is not considered a programming language. However, it is a huge part of web development and quite simple to learn. HTML allows the creation and structure of sections, paragraphs, and links using elements to build a web page. It is one of the most simple ways to begin programming since HTML runs on all web browsers and is easy to read!

What is CSS?

CSS or Cascading Style Sheets describes how HTML elements are to be displayed on screen, or in simple words: it provides style, color, and design to the page! Using HTML elements, CSS targets the elements we use to create our web pages to bring them to life by adding different formats, fonts, colors, animations, etc to the page. Many say if HTML is a house we are building. The HTML is the blueprint or skeleton of this house we are building, and the CSS is the decorating scheme that changes the look and feel.

What is JS?

JavaScript is a more simple programming/scripting language that web developers commonly use to create dynamic interactions when developing web pages, applications, servers, and games. JS adds practical and interactive features to a website such as buttons, textboxes, onClicks, etc. Rather than having static information, JS brings things to life with functions that allow you to update elements on the screen and get the user involved! To continue the house analogy, if HTML was the blueprint and CSS was the design, then JavaScript is the electrical side, or what keeps the elements running!

Where is this used?

These 3 tools are used to make this website and many of your other favorite web applications, games, sites, etc.

Where do I go?
